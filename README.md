# 자동차 경주 게임
## 요구 사항
### 조건
1. 주어진 횟수 동안, n대의 자동차는 `move forward` 또는 `stop`
2. 사용자는 몇 대의 자동차로 몇 번의 이동을 할 것인지를 입력할 수 있어야 한다.
3. `move forward` 조건
    - 0에서 9 사이에서 random 값을 구한다.
    - random 값이 4 이상일 경우 전진한다.
4. 자동차의 상태를 화면에 출력한다. 
    - 어느 시점에 출력할 것인지에 대한 제약은 없다.
    
### 구현할 기능 목록
1. 게임 시작
2. 명령 입력받기 (UI)
3. 전진 시도
4. 결과 출력하기 (UI)
